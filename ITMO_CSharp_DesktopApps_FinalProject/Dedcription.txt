Приложение позволяет просматривать данные и вносить изменения в таблицу [ShareDetails].[Shares] базы данных ApressFinancial,
разработанной в рамках курса “T-SQL и разработка MS SQL Server”.

При запуске приложения открывается диалоговое окно для авторизации.
Login: User1
Password: Qwerty123
Логин и пароль НЕ используются при подключении БД (при постановке задачи обсуждалось, что допустим такой подход).

Основной элемент управления – ToolStrip с несколькими кнопками, каждая из которых отвечает некоторому функциональному требованию 
(чтение и вывод на экран, удаление и добавление данных).

Область вывода данных – listView. 
В БД в таблице [ShareDetails].[Shares] 4 столбца. 
Приложение выводит на экран и позволяет задавать значения при вводе данных только для 3 столбцов: ShareDesc, ShareTickerID, CurrentPrice. 
Значения столбца ShareID (ключевое поле) в соответствии с настойками БД задаются автоматически. 
Предполагается, что эта информация не нужна пользователю и используется только для работы БД (ключевое поле, ограничение внешнего ключа для других таблиц).

1. Кнопка Show all выводит все элементы таблицы на экран.

2. Кнопка Find вызывает диалоговое окно, в котором можно задать критерии поиска данных по первым буквам ShareDesc и ShareTickerID. 
Если данные введены в оба текстовых поля диалогового окна, то поиск осуществляется с учетом логики «И». 
Если ни одно из текстовых полей формы поиска не заполнено, то выводятся все элементы таблицы.

3. Кнопка Add вызывает диалоговое окно для ввода нового элемента таблицы. 
Для текстовых полей, куда вводятся значения ShareDesc и CurrentPrice установлены обработчики событий Validating, 
поскольку эти поля не могут быть пустыми в соответствии с ограничениями, установленными в БД. 
Нет поля для ввода ShareID, т.к. это значение задается автоматически в соответствии с настройками БД. 

4. Кнопка Delete позволяет удалить одну или несколько строк из БД. 
Для этого нужно выделить одну или несколько строк в области вывода данных
(например, после вывода данных при нажатии кнопок Show all и Search) и нажать кнопку Delete. 

5. Кнопка Clear table позволяет пользователю очистить область вывода данных (для удобства).
